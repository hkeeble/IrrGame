enum IrrGame::GameState

Represents a game state.

Enumerator
READY 

The game is ready, but Run() has not been called.

RUNNING 

The game is currently running and is updating the main loop.

ERR_STATE 

The game is in error state, and will exit after the current loop.

EXIT 

The game is preparing to exit cleanly by user request, and will exit after the current loop.
